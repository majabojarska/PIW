# Laboratorium PIW

## Wykorzystane narzędzia

| Narzędzie         | Wersja               |
| :---------------- | :------------------- |
| Docker            | 19.03.7-ce           |
| System operacyjny | Manjaro Linux 19.0.2 |
| Twitter Bootstrap | 3.3.7                |

## Zbudowanie obrazu Dockera

`docker-compose build`

## Uruchomienie
Jeżeli obraz nie istnieje, zostaje zbudowany przed uruchomieniem.

`docker-compose up`
